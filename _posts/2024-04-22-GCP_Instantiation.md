## GCP 서버 생성 가이드 (1)

이 가이드는 Compute Engine을 이용해 인스턴스를 생성하고, SSH 접속을 설정하는 과정을 포함합니다.

#### 1. Compute Engine 인스턴스 생성

1. GCP 콘솔에 로그인합니다.
2. Compute Engine 메뉴로 이동한 후, "인스턴스 만들기"를 선택합니다.
3. 원하는 설정을 지정합니다:
   - 리전: 원하는 리전을 선택합니다. ~~서울 사람이라 서울로~~
   - 구성: 인스턴스 유형과 기타 설정을 선택합니다.
   - 운영체제: Ubuntu 20.04.6을 선택합니다.
   - 트래픽 허용: HTTP 및 HTTPS 트래픽 허용을 체크합니다.

4. 모든 설정을 완료한 후, "만들기" 버튼을 클릭하여 인스턴스를 생성합니다.
