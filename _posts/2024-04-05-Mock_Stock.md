### 모의 투자 서비스

#### 주요 기능 (Ver. 1)

- 계좌 발급
  - 가상 계좌 발급
  - 계좌 여러개 발급 가능
  - 기본 계좌 설정
- 금액 충전
  - 초기 접속 시 500만원 상당의 충천
  - 매일 접속 시마다 금액 충전
  - 친구 추천 시 금액 충전
- 주식 결제
- 구매한 주식 정보 조회
  - 주식 별 비중 (주 별, 금액 별)
  - 평단가. 구매 기록 (가격과 날짜. 거래 주 수)

#### 사용 예정인 기술 스택

- Spring Boot / Kotlin
- Prometheus / Grafana
- JPA / Querydsl
- Docker / Kubenetes / Nginx
- GCP
- MySQL / Redis
