### 커피 테이스팅 노트

#### 주요 기능 (Ver. 1)

- 일기 작성
  - 5단계 지표를 통해서 커피 맛 기록
  - 먹었던 카페 (집 // 구글 지도와의 연동)
  - 간단한 리뷰 작성 (사진과 글)
  - 카페의 경우 리뷰 공개 여부 설정
- 일기 조회
  - 날짜 기준 조회.
  - 원두를 통한 검색 기능 (ES)
- 원두 추천 받기
  - 내가 선호하는 원두를 기준으로
  - 추천하는 원두를 판매하는 카페
  - 구매할 수 있는 경로로 안내

#### 사용 예정인 기술 스택

- Spring Boot / Kotlin
- Prometheus / Grafana
- JPA / Querydsl
- Docker / Kubenetes / Nginx
- GCP
- MySQL / Redis
